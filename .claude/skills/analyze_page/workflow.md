# Workflow: analyze_page

Goal: analyze the webpage screenshot and explain what the page is about.

Steps:

1. Load the screenshot located at:

outputs/page_full.png

2. Observe the visual layout of the page.

3. Identify:

- page title
- main sections
- navigation areas
- article cards or headlines
- images and layout

4. Produce a structured explanation of the webpage.

Save the analysis to:

outputs/analysis/page_analysis.md
